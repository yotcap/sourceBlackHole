          Users can now edit account information via the myaccount page. sendmail is now included, allowing for a mailing list (which is an option on the register page). The readme file includes detailed information on all the files in the appliaction. Users also have the choice to have an e-mail sent to them if someone replies to one of their posts (using CDO, which can be changed to CDONTS if you need). name.inc controls all the names for the pages; this means that you only have to make changes in one file. The admin folder contains addcat and delcat for adding and delete categories. All I ask for letting you guys use this complete app, is that the "powered by: Atwinda Software" logo and link say at the bottom of every page.
