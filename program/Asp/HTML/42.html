ASP Online Quiz



          This is an Online quiz application written in both ASP and HTML. It uses MS Access database. It let users to undergo a quiz session, calculate marks, give the correct answer and compare user performance to their friends and many more.



ASP在线问答



          这是写在ASP和HTML的网上问答应用。 它采用MS Access数据库。 它让用户接受一个问答环节，计算出的痕迹，给出正确的答案，并比较用户性能给他们的朋友等等。
