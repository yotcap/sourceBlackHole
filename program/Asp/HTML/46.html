ASP Quicksite



          ASP Quicksite is basically a quick and easy site-in-a-box. Quicksite uses Macromedia Dreamweaver for templates, so all the design is up to you. ASP Quicksite has a Microsoft Access database backend, and includes the following features: user registration, log in, and log out; multi-category question and answer forum; and a comments system which allows for comments on any page. http://www.sau21.k12.nh.us/nhes/history is an example of the system in use
