Digger Solutions Intranet Open Source



          An Active Server Pages intranet for client, project, task, and timecard tracking and reporting as well as news, a calendar, a document repository, employee directory, discussion forum, surveys, and more! Fully customizable, easy to setup, and is completely functional right out of the box. Version 2.2 added multi-language support! Version 2.3 added support for international date formats! Version 2.3.2 adds a password reminder feature and makes auto-login of returning users and option. Version 2.4.0 adds project quotes and auto-generated work order numbers as well as some fixes for European number formats (x,xx rather than x.xx) Version 2.5.0 adds database and export utilities. Version 2.6 adds additional features and more admin utilities.
