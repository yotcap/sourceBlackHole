Seir Anphin

          Seir Anphin is a community oriented content management system. All html is stored in the database and users can choose between predefined template and colorsets. These can be created and imported/exported by administrators. There is a Shrine system which allows users to publish and maintain their own sections of the site. There is a built in message board which supports avatars, polls, quick and easy banning, file and image uploads, unlimited subforums, flood-checking, bbcodes, urlparsing and a personal notepad to save unfinished posts to for later. A rather unusual feature for such an application, it also has an rpg-inspired leveling system, where users gain semi-random amounts of experience for every post, topic and note. Other functions include a tagboard-like notes system, whose online, FAQs, memberlist, private messages, and perhaps most importantly, an extensive Administrative control panel for easy management.
