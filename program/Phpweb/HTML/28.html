          Jurpopage is Free Open Source Dynamic Website Scripting Project This first created and published at www.maya.or.id and now published to public in english at Jurpo.com It's coded under PHP and Mysql Database Server Project goal is to help others to create their own website with easy to changed webpage design This is one stop easy way to be a webmaster with your own touch the source code control. Jurpopage, share information easily through your own website. Jurpopage 0.1.0 released with new tinymce version (3_1_0_1) update, enjoy to post your content with richer HTML format and make your website content more life and friendly look.
