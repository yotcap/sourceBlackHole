JSystemsBB - UPDATED 24 august 2007 (beta3)


          JSystemsBB is a free, opensource bulletin board system, with most of features included: - Advanced session management system, forgot password menu, profile, upload avatar and user rank - Board statistics: most users online, online users now, total users, last user, total posts - Private messaging - Forum, topics icons (standard, popular, new topic/post) - Users list - You can set in admin panel: new theme, forum, sort themes, manage users, activate/deactivate users, posts sorting setup - Users can dlete/edit our posts setup - Users can upload avatar setup - Send wellcome email on users sign in, send generated password via email on forgot password - Installation routine included. Here you can set the admin name and moderator name, and other - After installation, you can sign in with admin name to be admin, and other user sign in with moderator name to be moderator - Different colours for administrator, moderator and standard users in posts, topics, statistics - and moore...
