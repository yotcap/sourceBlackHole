phpwind open source development framework windframework



          windframework is a lightweight php open source framework. It is the core components of the plant, providing like MVC, data persistence, view templates for applications such as engine technology. In addition windframework taken a component-based development model. Although the current robust component library is not too rich, but based on this development model, so that the expansion assembly becomes very easy.
