Flatfile CMS with many features


          Easy to remember urls Easy to install and configure Full featured file manager Easy to create new themes Search function Automatic page lists Installation is easy, simply copy the files to your server and chmod them to read/write/execute. After install you can make pages and upload files from within the administrator panel. Themes are very easy to create from a normal html file. The parameters are described in the readme file. A file manager is included in the administrator panel. It includes many functions useful in administering a website.
