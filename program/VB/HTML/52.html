Thumbnail Image Browser



          The ultimate Thumbnail Browser! Creates thumbnails of all VB 6.0 supported image formats from whatever folder it's pointed to. Uses BrowseForFolder API, so it will even work across networks. Click edit to shell your favorite editor and load the selected image. New code includes progress bar and saves the last window size size and position settings.
