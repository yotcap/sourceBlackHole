A completly 3d CAD program, updated again!



          This is a completly 3d CAD program! You can edit and create ANY 3d object and open & save them. With this program, all 3d objects of the Shot It game were made. You can create a 3d object and edit, rotate, move, connect, stretch... it. The program supports textures, materials, directional & indirectional light sources, setting the visibility, preview the 3d object and much more things. A word help file about this program is also included in the directory DX Cad\Help. This is the updated version which fixes some bugs and makes the exporting to the Shot It game much easier. This program needs DirectX 7 or higher to run. Check this out! Known bugs: In combination of Win2000 and DirectX 8, the textures in the preview are looking very strange. So, run the program on another PC configuration! Some people had a problem with loading the program because of an DirectDraw error.
