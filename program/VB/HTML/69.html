A Color Notepad



         This is a notepad that is black with green text... All the colors can be customized. It opens, saves (and save as), cuts, copies, pastes, and shows the file path (if saved or opened) on the Title bar. Demonstrates an easy way to save with a textbox... Also has horizontal and vertical scrollbars. I dont know the limit to the amount of text (if there is one).





颜色记事本



          这是一个记事本是黑绿色的文字...所有的颜色可以定制。 它打开，保存（保存为），割伤，复制，粘贴，并显示出文件路径的标题栏（如果保存或打开）。 演示一个简单的方法来保存一个文本框...还具有水平和垂直滚动条。 我不知道限制的文本量（如果有的话）
