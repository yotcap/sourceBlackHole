[Process Sniffer]


          This program lists all processes on computer and sorts them into listboxes depending on their detailed information. If it is available, you can also check detail info on a particular process, search for not-found processes. This application is meant to act like a Virus/Spyware/Adware protection that alerts you on common viruses etc. It automatically scans all your processes and matches them with the ones in database and those that are recorded as Spyware / Viruses etc, they are highlited with comment after the process name saying - ATTENTION This code is not fixing any viruses, spywares etc, it is only identifing the problem.
