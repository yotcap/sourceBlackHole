Image Editor



         An image editing program that currently supports Windows Bitmaps, PCX files, and my own attempt at image compression, the HCI. It has editing functions such as resizing/scaling with or without bilinear filtering, and a double size scanline feature; Rotation by any angle; Fitting the image to a wave or the surface of a sphere; and smooth filtering (to name a few).





图像编辑器



         目前支持Windows位图，PCX文件图像编辑程序，并在图像压缩我自己尝试，HCI。 它具有编辑功能，如调整/带或不带双线性过滤缩放，以及双尺寸扫描线特征; 通过旋转任意角度; 拟合图像到波或球体的表面上; 和平滑滤波（仅举几例）。
