A DBMS SYATEM IN C++



         A Database management system with password security provinding full freedom to create n number of databases with n number of table having n number of fields. Fields in the tables can be created by providing a certain data type such int,string,float etc. and the record entry is bound to that data type i.e one can not enter a string in a integer field. The system also provides a facility to edit,view,delete the records. It's a Hierarchical style system that first creates a database then it creates tables and finally the fields.
