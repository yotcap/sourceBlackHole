MENU MAKER --- creates menus for your applications.through a wizard


          in C or C++ you have to waste valuable time plugging in code to create a nice menu using arrow keys and stuff. This program is the answer to that. When you compile and run the program it asks you a few simple questions, it runs like an office wizard asking a few simple questions. Depending on your answers it generates a header file menu.h which you can include at the top of your program with a simple function call menu().
