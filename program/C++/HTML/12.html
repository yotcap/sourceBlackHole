A cool FileServer



          You will see, this is actually two apps using Winsock. The one is the file server and the other is the client. With the client you can specify a file to download from the server. It then downloads the file with a speed between 250 - 360 kb/s depending on your network (I could only test it on a LAN)Other cool thing: Multithreading for multiple connections and when you minimize the server dialog, it jumps to the system tray.
