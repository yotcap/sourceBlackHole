Dictionary


          This is a complete application. It was compiled using Turbo C++ 3.0. This is an English Dictionary with a very large vocabulary. The project has an installation program (install.cpp). During the installation, all the necessary helps will appear at the screen. Installation may take a long time, please be patient! After install find the word/words as you wish and you will be astonished how large its vocabulary is. Searching of words is very easy and in a user friendly manner.
