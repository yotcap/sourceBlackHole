Gather as much as possible information about a PC in the internet



          This app is an IP range scanner. It gathers informations like: SMB shares, services, processes, transport protocols, number of drives, groups, users, sessions, remote registry info and much more... It's also possible to shutdown a PC, to open shares and to send messages over the message service.
