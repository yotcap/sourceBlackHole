Animated Movie Maker



          This application provides an easy animated movie maker. The code is an improved version of my capture picture program, which I uploaded a long time ago. I decided to make it a little more interesting and this time I provided a lot of comments. :-) After choosing the right driver, filename and framerate, you can record single frames from any videodevice such as a webcam into a video file and make your own animated movie. (*.avi). For example put your webcam on a steady place and focus it on a little figure. Record a view frames and move the figure just a little bit. Repeat this a view times and you will have an animated movie. This application also shows how to use C++ for programming windows without using the MFC. The user interface consists of a modeless dialog window without creating a windowclass or parent window. All video functions are realized with Video For Windows functions. In order to use this functions you must include the video library vfw32.lib which must be entered into the linker settings.
